# DESIGN.md - Developer Portfolio Theme

## Overview
Dark, developer-oriented GitHub Pages portfolio for GeumSang Kang.

## Design goals
- Developer feeling with terminal-style card, dark navy background, grid texture, cyan/blue accents.
- Simple static HTML/CSS, no backend and no paid external API.
- Responsive layout for desktop and mobile.
- Accessible navigation with links to About, Projects, CV, Courses, AI Usage, and Contact.

## Palette
- Background: #020617
- Panel: #0f172a
- Text: #e5e7eb
- Muted: #94a3b8
- Accent: #38bdf8
- Blue: #2563eb
- Terminal prompt green: #22c55e

## Content checklist
Personal introduction, profile photo, contact information, CV/resume, two projects/activities, navigation menu, planned courses, and AI usage section.
